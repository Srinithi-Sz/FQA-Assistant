# FQA Assistant 🤖

FQA Assistant is a lightweight chatbot built using HTML, CSS, and JavaScript.  
It answers Computer Science related questions using a keyword-based similarity matching approach.

---

## 📌 Project Overview

FQA Assistant is designed to respond to frequently asked Computer Science questions.  
The system processes user input, compares it with predefined FAQ questions, and returns the most relevant answer.

This project demonstrates basic chatbot logic, text similarity techniques, and modern UI design.

---

## 🚀 Features

- Keyword-based similarity matching
- Dynamic typing animation effect
- Dark glassmorphism UI
- Enter key support
- CSE-based FAQ dataset
- Responsive design

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---

## ⚙️ Working Principle

1. User enters a question.
2. The input is converted to lowercase.
3. The system compares the question with stored FAQs.
4. A similarity score is calculated.
5. The best matching answer is displayed with a typing effect.
